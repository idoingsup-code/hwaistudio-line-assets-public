# HW AI Studio LINE 素材助手 / 素材管理器

版本：v0.9.19 收藏權限歸屬與收藏列版  
更新日期：2026-05-28  
專案：LINE 外掛貼圖 / HW AI Studio LINE 素材助手  
狀態：PNG/JPG 貼圖流程保留；GIF 動態素材可切換模式；收藏改為使用者端個人設定；MP4/MOV/M4V 已移除，影片請用 LINE 內建附件。

## 1. 本版結論

v0.9.19 主要修正收藏權限歸屬與使用者端收藏操作。

左側選單改為「基本操作 / 資料 / 資訊」，右側選單改為「貼上行為設定」。這樣可避免左右選單重複，也讓指定 LINE、選擇資料夾、重新載入素材等功能集中在同一處。

GIF / APNG 新增兩種模式：

```text
動態圖片貼上：保動畫模式
動態圖片貼上：直接圖片模式
```

「保動畫模式」維持原本可用流程；「直接圖片模式」會把 GIF 第一幀當成圖片貼出，流程較乾淨，但可能不保留動畫。

---

## v0.9.19 補充：收藏權限歸屬

收藏與最近使用屬於使用者端個人資料，不屬於官方公開素材包。

管理端不再顯示或管理「收藏 / 最近使用」素材包；管理端只負責官方素材包的匯入、標題、標籤、排序、啟用、檢查與發布。

使用者端每張素材下方固定顯示收藏列：

```text
☆ 未收藏
★ 已收藏
```

操作規則：

```text
點圖片本體 → 貼到 LINE
點 ☆ 未收藏 → 加入收藏
點 ★ 已收藏 → 取消收藏
點上方星星分頁 → 查看已收藏素材
```

收藏資料只儲存在使用者本機 user_state，不會發布到官方公開 repo。

## 2. 日常使用入口

根目錄保留三個主要入口：

```text
01_START_LINE_ASSISTANT.bat
02_START_ASSET_MANAGER.bat
03_PUBLISH_OFFICIAL_ASSETS.bat
```

### 01_START_LINE_ASSISTANT.bat

開啟使用者端。用途：選擇 PNG/JPG 貼圖與 GIF 動態素材並送到 LINE。

### 02_START_ASSET_MANAGER.bat

開啟管理端。用途：匯入、編輯、刪除、檢查、發布素材。

### 03_PUBLISH_OFFICIAL_ASSETS.bat

一鍵發布官方公開素材。發布時會同步主目錄 `README.md` 到官方公開 repo。

```text
動態圖片貼上：直接圖片模式
```

---

## 4. 素材貼上規則

### PNG / JPG

走貼圖模式。預設使用純圖片剪貼簿，目標是讓 LINE 直接吃圖片，不顯示多餘文字選項。

### GIF / APNG

走動態圖片模式，可由右側選單切換：

```text
保動畫模式：維持原本可用流程，LINE 可能顯示「圖片 / 文字」選擇。
直接圖片模式：擷取第一幀貼成圖片，較乾淨，但可能不保留動畫。
```

### MP4 / MOV / M4V

v0.9.17 起已從本工具移除。影片請使用 LINE 內建附件功能處理。

## 5. 正確發布目標

官方 LINE 素材公開 repo 必須是：

```text

```

對應遠端 URL：

```text
https://github.com/idoingsup-code/hwaistudio-line-assets-public.git
```

不可發布到：

```text
hwaistudio-public-assets
```
